## üõ†Ô∏è Setup & Installation

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ishadowprince716/personal-portfolio-website.git
   cd personal-portfolio-website
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   ```

3. **View in browser**
   - Navigate to `http://localhost:8000` (or the port shown)

### GitHub Pages Deployment

1. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```

2. **Enable GitHub Pages**
   - Go to repository Settings
   - Navigate to Pages section
   - Select source: `main` branch
   - Click Save
   - Your site will be live at: `https://ishadowprince716.github.io/personal-portfolio-website/`

## üé® Customization

### Update Personal Information

1. **Edit `index.html`**
   - Update name, title, and description in hero section
   - Add your projects in the projects section
   - Update contact information
   - Add your social media links

2. **Customize Colors**
   - Edit CSS variables in `styles.css` under `:root`
   ```css
   :root {
       --primary-color: #667eea;     /* Change primary color */
       --secondary-color: #764ba2;   /* Change secondary color */
       --accent-color: #f093fb;      /* Change accent color */
   }
   ```

3. **Add Profile Image**
   - Replace placeholder image URL in `index.html`
   - Add your actual image to the project and link it

4. **Update Projects**
   - Add your actual project details
   - Include project images
   - Update GitHub and demo links
